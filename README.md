## Installation

Install my-project with npm

```bash
  composer require rajtechnologies/laravel-route-list
```

Go On Web Brower 
```
  {base_url}/route-list
```
    
## Features

- Web Route List
- API Route List
- filter with Datatable of method, route , name and action
